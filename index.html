Showcases

Following sections display some of my previous work.

Oreowolves    Visit Website

Oreowolves is an online version of the popular game Werewolves Kill. It features Google OAuth, lobby system, in game chat, and computer host. The inspiration came after a game with friends went chaos when the host person made an irreversible error. Therefore, I thought to myself, why not let the computer do the work?

Scavenger

Scavenger is an IOS game. The theme is to combine Congra and Little Fighter 2 into a mobile arcade version. Joystick can be triggered by swipe and hold anywhere on the screen, tap on either sides of the character to shoot. UnityEngine C# is the tool used to build this game. Special thanks to Cliff Chen for drawing the graphics for this game.

Find My Study Website    Visit Website

Find My Study is an online search, aggregation and listing engine for international students and their families. The majority of my work was to develop the advertisement and FAQ engine using C# MVC. The site was launched in September 2016.

Cocomap

Cocomap is a collaborative concept map drawing tool. It can analyze and merge all individuals' concept maps into a single, enhanced version for display. The purpose of Cocomap is to improve study efficiency. It is built upon Meteor and ReactJs.

SkyPeak    

SkyPeak is a flying simulation game built upon Unity C#. The goal of the game is to collect letters and deliver them in different environments.

  • Features:
  • 3D flying
  • Different Scenes
  • Multiplayers

Mushroom Shooter

Mushroom Shooter is a PC mini-game available in Microsoft Store. The objective of this game is to shoot out all the mushrooms as quick as possible. The game was developed on Construct 2.

HighFlyer

Highflyer is a PC mini-game available in Microsoft Store. The objective of this game is to dodge the obstacles and fly for as long as possible. The game was developed on Construct 2.
